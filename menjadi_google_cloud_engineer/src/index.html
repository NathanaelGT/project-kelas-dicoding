Web Animeh

A self-hosted web-based anime download and streaming application. Using a scraper to get data from the fansub website and synchronise it with data from MyAnimeList. use.

As a web-based application, this project divides the work between the client and the server. The server will scrape the fansub and then store the validated data into the database, the data will then be updated using MyAnimeList to get the latest updates. The client is responsible for displaying all the data in realtime to the user. The client can also request the server to download the selected anime episode for later use.

SIAMEL

SIAMEL is an e-learning website and student academic information system. This website aims to make it easier for staff, lecturers, and students to take care of academic needs.

The website includes management systems for faculty, study programs, buildings, staff, professors, students, courses, classes, and classrooms to be managed by staff. In addition, there are also assignments, meeting schedules, attendance, study groups, and assessments that can be managed by the lecturers. Finally, students can make course selections, submit assignments, manage study groups if permitted, and view final assessment results.
